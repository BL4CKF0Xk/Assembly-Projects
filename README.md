### Usage

You wan't to install "nasm" for compile program

after install nasm type this in your terminal

```bash
    nasm -f elf64 -o Hello.asm Hello.o
```
    
Next you want to install linker. after install that typr this on your terminal
```bash
    ld -o Hello.o Hello
```
      
And run the Program!
```bash
    ./Hello
```
